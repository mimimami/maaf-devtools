# Changelog

## [1.0.0] - 2024-01-XX

### Added

- ✅ **Request Inspector**
  - `RequestEntry` - Request bejegyzés Telescope szerű inspector-hoz
  - `RequestInspector` - Request inspector Telescope szerű funkcionalitással
  - Request/response tracking
  - Query, log, event tracking per request
  - Duration és memory tracking

- ✅ **Module Log View**
  - `ModuleLogView` - Modulonkénti log nézet
  - Module-level logging
  - Log level filtering
  - File-based logging

- ✅ **Query Profiler**
  - `QueryProfiler` - Query profiler adatbázis lekérdezésekhez
  - `QueryEntry` - Query bejegyzés profiler-hez
  - Query duration tracking
  - Slow query detection
  - Module-level query tracking

- ✅ **Dashboard**
  - `DevToolsDashboard` - DevTools dashboard összes adattal
  - Request inspector view
  - Module log view
  - Query profiler view
  - HTML dashboard template

- ✅ **CLI Commands**
  - `DevToolsDashboardCommand` - DevTools dashboard generálása
  - `DevToolsStatsCommand` - DevTools statisztikák megjelenítése
